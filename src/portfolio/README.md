# Arun Narangali – Portfolio Workspace

Welcome to my portfolio workspace. Each file represents a focused area of my experience, and each tab shows how I approach delivery and impact.
Browse through the sections to understand my skills, projects, and the outcomes I have helped teams achieve.

---

## 🚀 Quick Start

1) Open the **Explorer** and select a file like `Home.tsx` or `Projects.tsx`.
2) Press **Cmd/Ctrl+Shift+P** to open the Command Palette.
3) Use **Search** from the Activity Bar to find skills, projects, or experience.
4) Toggle the **Terminal** from the top bar to run commands.

---

## 🧭 UI Map

- **Activity Bar**
  - Explorer: file tree + open editors
  - Search: global content search
  - Source Control: career journey + changelog
  - Extensions: skill/tool cards

- **Left Panel**
  - **Explorer** lists all portfolio files and active tabs
  - **Search** shows results across projects, skills, and experience
  - **Source Control** shows career journey + changelog
  - **Extensions** shows tools and proficiency badges

- **Editor Area**
  - Tabs for open files
  - Preview or code view (for TSX pages)
  - Split editor (desktop only)

- **Top Bar**
  - Terminal toggle
  - Split editor toggle
  - Preview ↔ Code toggle
  - More actions menu (links, resume, share)

- **Bottom Status Bar**
  - Repo shortcut + quick links (React, TypeScript, Node)

## ⚡ Actions & Shortcuts

- **Command Palette**: `Cmd/Ctrl+Shift+P`, filter commands, navigate with ↑/↓, run with Enter
- **Close Palette**: `Esc`
- **Quick Shortcuts**: `Cmd/Ctrl+Shift+Alt+F` (Find), `Cmd/Ctrl+Shift+Alt+T` (Theme), `Ctrl+`` (Terminal), `Cmd/Ctrl+Shift+Alt+B` (Toggle Sidebar), `Cmd/Ctrl+Shift+Alt+L` (Format)
- **Note**: Some VS Code defaults are browser-reserved; web-safe equivalents are listed here.
- **Preview ↔ Code**: toggle from the top bar or editor group
- **Split Editor**: desktop-only split from the top bar
- **Search**: Activity Bar → Search icon

## 🎨 Theme & Layout

- **Settings modal** (gear icon in Activity Bar)
  - Theme: Dark+, Light+, High Contrast
  - Layout: Comfortable or Compact
- **Editor mode**: Single page (default ON) lets you scroll all sections while tabs follow automatically; code is view-only
- **Terminal shortcuts**
  - `theme dark` / `theme light` / `theme highContrast`
  - `layout comfortable` / `layout compact`

---

## 📄 Resume & Share Links

- **Download Resume (PDF)**
  - Title bar menu → “Download Resume (PDF)”
  - Terminal command: `resume`
  - Settings modal includes a Resume button

- **Share or Copy Portfolio**
  - Title bar menu → “Copy Portfolio URL” or “Share Portfolio”
  - Terminal command: `url` or `copy url`

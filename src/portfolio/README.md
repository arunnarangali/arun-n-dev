# Arun Narangali – Portfolio Workspace

This workspace mirrors the sections rendered inside the VS Code-style portfolio.

## Sections

- `Home.tsx`: hero headline, metrics, CTAs
- `About.tsx`: bio card + background
- `Projects.tsx`: selected case studies
- `Experience.tsx`: timeline entries
- `Skills.tsx`: stack grid
- `Contact.tsx`: contact form and social links

## Principles

- React 18 + TypeScript strict mode
- Tailwind utility surface with custom tokens
- Accessibility-first (focus states, contrast, semantic groups)
- Minimal, deliberate motion (no neon glows)
